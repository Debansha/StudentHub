🎓 StudentHub

A web-based academic management platform for managing students, faculty, courses, attendance, grades, and academic records. Built using PHP, MySQL, and Bootstrap, the application provides role-based access for administrators, faculty, and students with features such as attendance tracking, GPA/CGPA calculation, PDF report generation, and performance monitoring.

✨ Features
Role-based login (Admin, Faculty, Student)
Student, Faculty, Course & Department Management
Attendance and Marks Management
Automatic GPA/CGPA & Grade Calculation
PDF Report Card Generation
Placement & Scholarship Eligibility Check
Secure Authentication with PHP Sessions


🛠️ Tech Stack
PHP 8
MySQL
HTML5, CSS3, Bootstrap 5
JavaScript
Apache (XAMPP)
FPDF
